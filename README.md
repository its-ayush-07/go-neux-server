# GoNeux Server

Technologies: Go, PostreSQL, JWT, gin, gorm

### Installation

- Fork and clone
- Install the packages and dependencies
```
go mod tidy
```
- Run the server
```
go run main.go
```

### Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`CLIENT_ORIGIN`

`DB_URL`

`SENDER_EMAIL`

`SENDER_PASSWORD`

`SECRET_KEY`

`COOKIE_DOMAIN`
